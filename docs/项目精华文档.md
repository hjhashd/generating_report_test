# 🚀 报告生成系统 - 精华文档

## 📋 项目概述

**generate_report_test** 是一个基于 FastAPI 的智能报告生成平台，集成 AI 辅助功能，支持文档处理、报告合并、用户管理和权限控制。

---

## 🎯 核心功能

### 🔧 主要模块
- **报告管理**: 上传、查询、删除、合并报告
- **AI 辅助**: 智能摘要、文本优化、内容生成
- **文档处理**: Word/PDF 格式支持，目录树管理
- **用户系统**: JWT 认证，角色权限控制

### 🌐 API 服务
- **端口**: 34521 (开发) / 12543 (生产)
- **框架**: FastAPI + Uvicorn
- **认证**: JWT Token

## ⚠️ 重要注意事项

### 🔐 安全配置
- JWT 密钥必须保密

- 确保数据库服务器可访问
- AI 服务地址需要正确配置
- 防火墙开放相应端口

## 🆘 故障排除

### 🔧 常见问题

1. **端口占用**
   ```bash
   # 检查端口占用
   lsof -i :34521
   # 强制释放端口
   kill -9 <PID>
   ```

2. **数据库连接失败**
   - 检查数据库服务器状态
   - 验证数据库连接信息
   - 确认网络可达性

3. **容器启动失败**
   ```bash
   # 查看容器状态
   docker ps -a
   # 查看详细日志
   docker logs <container_id>
   ```
